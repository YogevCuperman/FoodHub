Welcome to FoodHub!

This is an explanation of how to run the app:


## Prerequisites

Before you can run this web app, ensure you have the following dependencies installed on your system:

- Node.js (v14.0 or higher)
- npm (v6.0 or higher)


## Installation

1. Navigate to the project directory:
    bash command:
        cd /path/to/the/project/directory

2. Install the required npm packages as described in the package.json file:
    bash command:
        npm install


## Usage

1. Start the Express.js server:
    bash command:
        npm start

2. Open a web browser and navigate to: http://localhost:3000

3. Enjoy!
